# ConFin

Aplicación personal de finanzas en formato PWA.

## Funciones
- Ingresos, gastos y transferencias
- Cuentas
- Presupuestos mensuales
- Metas de ahorro
- Resumen por categoría
- Exportar e importar respaldo JSON
- Modo oscuro optimizado para iPhone
- Funciona sin conexión después de la primera carga

## Publicar gratis con GitHub Pages
1. Crea un repositorio público en GitHub llamado `confin`.
2. Sube todos los archivos de esta carpeta a la raíz del repositorio.
3. En GitHub entra a **Settings > Pages**.
4. En **Build and deployment**, selecciona **Deploy from a branch**.
5. Elige la rama `main` y la carpeta `/ (root)`.
6. Guarda. GitHub mostrará la dirección pública.

## Instalar en iPhone
1. Abre la dirección publicada en Safari.
2. Pulsa Compartir.
3. Selecciona **Agregar a pantalla de inicio**.
4. Activa **Abrir como app web** si aparece.
5. Pulsa **Agregar**.

## Privacidad
Los datos se almacenan en el navegador del dispositivo. Usa periódicamente **Ajustes > Exportar respaldo**.
